## Push Notifications

### Firebase Cloud Messaging Setup

**Notification Types:**
1. `transaction_pending` - New transaction needs confirmation
2. `payment_reminder` - Payment due reminder (3 days, 1 day, due day)
3. `payment_success` - Payment received confirmation
4. `credit_alert` - Credit limit alerts
5. `promotion` - Promotional messages

**API Endpoint (NEW):**
```
POST /api/v1/customers/me/devices
Body: { "fcm_token": "string", "device_type": "ios|android", "device_name": "string" }

DELETE /api/v1/customers/me/devices/{device_id}
```
